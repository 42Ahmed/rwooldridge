18.5 C5 [INTQRT]

Use INTQRT for this exercise. (i) In Example 18.7, we estimated an error correction model for the holding yield on six-month T-bills, where one lag of the holding yield on three-month T-bills is the explanatory variable. We assumed that the cointegration parameter was one in the equation \(hy6_t = \alpha + \beta hy3_{t-1} + u_t\). Now, add the lead change, \(\Delta hy3_t\), the contemporaneous change, \(\Delta hy3_{t-1}\), and the lagged change, \(\Delta hy3_{t-2}\), of \(\Delta hy3_{t-1}\). That is, estimate the equation \[hy6_t = \alpha + \beta hy3_{t-1} + \phi_0 \Delta hy3_t + \phi_1 \Delta hy3_{t-1} + \rho_1 \Delta hy3_{t-2} + e_t\] and report the results in equation form. Test H\(_0\): \(\beta = 1\) against a two-sided alternative. Assume that the lead and lag are sufficient so that \(\{hy3_{t-1}\}\) is strictly exogenous in this equation and do not worry about serial correlation. (ii) To the error correction model in (18.39), add \(\Delta hy3_{t-2}\) and \((hy6_{t-2} - hy3_{t-3})\). Are these terms jointly significant? What do you conclude about the appropriate error correction model?

18.8 C8 [FERTIL3]

(ii) Using the data through 1979, estimate a cubic time trend model for gfr (that is, regress gfr on \(t\), \(t^2\), and \(t^3\), along with an intercept). Comment on the \(R\)-squared of the regression. (iii) Using the model in part (ii), compute the mean absolute error of the one-step-ahead forecast errors for the years 1980 through 1984. (iv) Using the data through 1979, regress \(\Delta gfr_t\) on a constant only. Is the constant statistically different from zero? Does it make sense to assume that any drift term is zero, if we assume that \(gfr_t\) follows a random walk?

  1. Now, forecast \(gfr\) for 1980 through 1984, using a random walk model: the forecast of \(gfr_{n+1}\) is simply \(gfr_n\). Find the MAE. How does it compare with the MAE from part (iii)? Which method of forecasting do you prefer?
  1. Now, estimate an AR(2) model for \(gfr\), again using the data only through 1979. Is the second lag significant?
  2. Obtain the MAE for 1980 through 1984, using the AR(2) model. Does this more general model work better out-of-sample than the random walk model?

18.15 C15 [BEVERIDGE]

This question asks you to study the so-called Beveridge Curve from the perspective of cointegration analysis. The U.S. monthly data from December 2000 through February 2012 are in BEVERIDGE. (i) Test for a unit root in \(urate\) using the usual Dickey-Fuller test (with a constant) and the augmented DF with two lags of \(curate\). What do you conclude? Are the lags of \(curate\) in the augmented DF test statistically significant? Does it matter to the outcome of the unit root test? (ii) Repeat part (i) but with the vacancy rate, \(vrate\). (iii) Assuming that \(urate\) and \(vrate\) are both I(1), the Beveridge Curve, \[urate_t = \alpha + \beta vrate + u_t,\] only makes sense if \(urate\) and \(vrate\) are cointegrated (with cointegrating parameter \(\beta < 0\)). Test for cointegration using the Engle-Granger test with no lags. Are \(urate\) and \(vrate\) cointegrated at the 10% significance level? What about at the 5% level? (iv) Obtain the leads and lags estimator with \(cvrate_t\), \(cvrate_{t-1}\), and \(cvrate_{t+1}\) as the I(0) explanatory variables added to the equation in part (iii). Obtain the Newey-West standard error for \(\hat{\beta}\) using four lags (so \(g = 4\) in the notation of Section 12-5). What is the resulting 95% confidence interval for \(\beta\)? How does it compare with the confidence interval that is not robust to serial correlation (or heteroskedasticity)? (v) Redo the Engle-Granger test but with two lags in the augmented DF regression. What happens? What do you conclude about the robustness of the claim that \(urate\) and \(vrate\) are cointegrated?
