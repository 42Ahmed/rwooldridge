compute the LM statistic for testing whether motheduc and fatheduc are jointly significant. \[\begin{align}bwght =& \beta_0 + \beta_1 cigs + \beta_2 parity + \beta_3 faminc \\&+ \beta_4 motheduc + \beta_5 fatheduc + u\end{align}\]

The Math Way:

To do Lagrange Multiplier Test:

  1. Filter the data from missing data
  2. Estimate the restricted model
  3. Extract the residuals \(\tilde u\)
  4. Regress \(\tilde u\) on all independent variables
  5. Extract \(R^2_u\) and \(n\)
  6. \(LM = n \cdot R^2_u\)
  7. \(LM \sim \chi^2_q\) where q is number of restrictions
df_clean <- df_bwght |>
  drop_na(bwght, cigs, parity, faminc, motheduc, fatheduc)

model_r <- lm(bwght ~ cigs + parity + faminc,
  data = df_clean
)

df_clean$u_tilde <- resid(model_r)

model_aux <- lm(
  u_tilde ~ cigs + parity + faminc + motheduc + fatheduc,
  data = df_clean
)

n <- nobs(model_aux)
r2_aux <- summary(model_aux)$r.squared

lm_stat <- n * r2_aux
p_val <- pchisq(lm_stat, df = 2, lower.tail = FALSE)

cat("LM Statistic: ", lm_stat, "\n")
LM Statistic:  2.882101 
cat("P-value:      ", p_val, "\n")
P-value:       0.236679

Interpretation:

Calculated LM Statistic is 2.8 with p value 0.24. We fail to reject the null hypothesis; education is not jointly statistically significant.

Important

If we wanted to test if education = 0.5 not just zero, we will modify y from step two

df_clean <- df_clean |> mutate(bwght_tilde = bwght - (0.5 * motheduc) - (0.5 * fatheduc))

5.4 C4 [SKEWNESS]

Several statistics are commonly used to detect nonnormality in underlying population distributions. Here we will study one that measures the amount of skewness in a distribution. Recall that any normally distributed random variable is symmetric about its mean; therefore, if we standardize a symmetrically distributed random variable, say \(z = (y - \mu_y)/\sigma_y\), where \(\mu_y = \text{E}(y)\) and \(\sigma_y = \text{sd}(y)\), then \(z\) has mean zero, variance one, and \(\text{E}(z^3) = 0\). Given a sample of data \(\{y_i: i = 1, \dots, n\}\), we can standardize \(y_i\) in the sample by using \(z_i = (y_i - \hat{\mu}_y)/\hat{\sigma}_y\), where \(\hat{\mu}_y\) is the sample mean and \(\hat{\sigma}_y\) is the sample standard deviation. (We ignore the fact that these are estimates based on the sample.) A sample statistic that measures skewness is \(n^{-1} \sum_{i=1}^n z_i^3\), or where \(n\) is replaced with \((n - 1)\) as a degrees-of-freedom adjustment. If \(y\) has a normal distribution in the population, the skewness measure in the sample for the standardized values should not differ significantly from zero.

  1. First use the data set 401KSUBS, keeping only observations with fsize = 1. Find the skewness measure for inc. Do the same for log(inc). Which variable has more skewness and therefore seems less likely to be normally distributed?
  1. Next use BWGHT2. Find the skewness measures for bwght and log(bwght). What do you conclude?
  1. Evaluate the following statement: “The logarithmic transformation always makes a positive variable look more nearly normally distributed.”
  1. If we are interested in the normality assumption in the context of regression, should we be evaluating the unconditional distributions of \(y\) and \(\log(y)\)? Explain.
