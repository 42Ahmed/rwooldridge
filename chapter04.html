The Two Times Rule

If the absolute estimate is at least **twice as large** as its standard error, it is statistically significant

  1. In terms of the parameters, state the null hypothesis that a 1% increase in A’s expenditures is offset by a 1% increase in B’s expenditures.

Interpretation:

\[\begin{align} H_0: \beta_1 + \beta_2 &= 0\\ H_1: \beta_1 + \beta_2 &\neq 0 \end{align}\]

  1. Estimate the given model using the data in VOTE1 and report the results in usual form. Do A’s expenditures affect the outcome? What about B’s expenditures? Can you use these results to test the hypothesis in part (ii)?

Interpretation:

All the independent variables are statistically significant, indicating that \(\beta \neq 0\). We still can’t test the hypothesis from part 2 because we don’t have the covariance needed

\[ se(\hat \beta_1 + \hat \beta_2) = \sqrt{Var(\hat \beta_1) + Var(\hat \beta_2) + 2 Cov(\hat \beta_1, \hat \beta_2)} \]

  1. Estimate a model that directly gives the t statistic for testing the hypothesis in part (ii). What do you conclude? (Use a two-sided alternative.)

The easy way:

joint_test <- linearHypothesis(model_vote1,
                               "lexpendA + lexpendB = 0")
print(joint_test)

Linear hypothesis test:

Model 1: restricted model
Model 2: voteA ~ lexpendA + lexpendB + prtystrA

  Res.Df   RSS Df Sum of Sq      F Pr(>F)
1    170 10111                           
2    169 10052  1    59.261 0.9963 0.3196

Interpretation:

p > 0.05, we fail to reject the hypothesis; if both candidates double their budget, election outcome stays the same.

The math way:

We define a new estimator \(\theta = \beta_1 + \beta_2\) which means that \(\beta_1 = \theta - \beta_2\) Substitute into the original equation: \[ \begin{align} voteA &= \beta_0 + \beta_1 \log(expendA) + \beta_2 \log(expendB) + \beta_3 prtystrA + u \\ voteA &= \beta_0 + (\theta - \beta_2) \log(expendA) + \beta_2 \log(expendB) + \beta_3 prtystrA + u \\ voteA &= \beta_0 + \theta \log(expendA) - \beta_2 \log(expendA) + \beta_2 \log(expendB) + \beta_3 prtystrA + u \\ voteA &= \beta_0 + \theta \log(expendA) + \beta_2[\log(expendB) - \log(expendA)] + \beta_3 prtystrA + u \end{align}\]

model_vote1_reparameterized <- lm(
  voteA ~ lexpendA + I(lexpendB - lexpendA) + prtystrA, data = df_vote1)

Interpretation:

Adding a 150 square foot bedroom is associated with an estimated 8.58% increase in price. We are 95% sure the true value lies in the interval [3.33%, 13.83%]

  1. write \(\beta_2\) in terms of \(\theta\) and \(\beta_1\)

The Math Way:

To get the confidence Interval for a 150 square foot bedroom, let \(\theta = 150 \times \beta_1 + \beta_2\).

\[ \begin{align} \beta_2 &= \theta - 150 \beta_1\\ \log(price) &= \beta_0 + \beta_1(sqrft) + (\theta - 150 \beta_1)bdrms + u \\ \log(price) &= \beta_0 + \beta_1 (sqrft - 150 \cdot bdrms) + \theta(bdrms) + u \end{align} \]

  1. Obtain standard error for \(\hat \theta\) and construct a 95% confidence Interval

We run regression to obtain its standard error and confidence interval

df_hprice1$sqrft_adj <- df_hprice1$sqrft - 150 * df_hprice1$bdrms

model_theta <- lm(I(100*lprice) ~ sqrft_adj + bdrms, data = df_hprice1)




cm <- c("bdrms" = "Effect of 150 sqft Bedroom (%)")

modelsummary(
  model_theta,
  coef_map = cm,
  fmt = 2,
  estimate = "{estimate} [{conf.low}, {conf.high}]",
  statistic = NULL,   
  title = "Estimated Impact of Adding a 150 sqft Bedroom",
  gof_omit = ".*"     
)
Estimated Impact of Adding a 150 sqft Bedroom
(1)
Effect of 150 sqft Bedroom (%) 8.58 [3.26, 13.90]

Interpretation:

We scaled dependent variable by 100 to get percentage values for the log-level model. The estimated effect is 8.58% with a confidence interval of [3.26%, 13.90%].

Caution

deltMethod relies on asymptotic theory, confidence intervals are calculated using Z distribution. Critical value = 1.96

lm uses t distribution. With 85 degrees of freedom, critical value = 1.988
